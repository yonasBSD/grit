## What changed

New shared engine `grit-lib/src/signing.rs` (port of git `gpg-interface.c` +
`commit.c`):
- `GpgConfig::from_config` reads `gpg.format` (case-sensitive: `openpgp` ok,
  `OpEnPgP` errors), `gpg.<fmt>.program`/`gpg.program`, `user.signingkey`,
  `gpg.minTrustLevel`; resolves the program (`~` -> `$HOME`, absolute, bare name
  on `$PATH`).
- `sign_buffer` spawns `<program> --status-fd=2 -bsau <key>`, captures the
  armored detached signature, and fails (surfacing stderr) on non-zero exit or
  missing `SIG_CREATED` — the `LET_GPG_PROGRAM_FAIL`/`zOMG` path.
- `add_header_signature` splices the `gpgsig` header (insert at first `\n\n`+1,
  `gpgsig ` then space-prefixed continuation lines).
- `extract_signed_payload` + `verify_commit` rebuild the signed payload, run
  `<program> --status-fd=1 --keyid-format=long --verify <sigfile> -`, and parse
  `[GNUPG:]` status lines (GOODSIG/BADSIG/ERRSIG/EXPSIG/EXPKEYSIG/REVKEYSIG/
  VALIDSIG/TRUST_*) into a `SignatureCheck` exposing result/trust/key/signer/
  fingerprint/primary-fingerprint/raw-status.
- `grit-lib/src/error.rs`: new `Error::Signing`. `lib.rs`: `pub mod signing;`.

CLI wiring:
- `commit.rs`: `-S`/`--gpg-sign[=KEY]`/`--no-gpg-sign` now sign the commit
  (honors `commit.gpgsign`); failure aborts the commit.
- `commit_tree.rs`: adds `-S`/`--gpg-sign[=KEY]`/`--no-gpg-sign`; signs only on
  explicit request (bare `commit-tree` ignores `commit.gpgsign`, matching git).
- `merge.rs`: fixed `-S` (was wrongly aliased to `--signoff`) to mean
  `--gpg-sign`; signs the merge commit across all merge-commit creation paths
  (real/octopus/ours/theirs). `pull.rs` forwards the new fields.
- `verify_commit.rs`: real verification — gpg output to stderr (`--raw` for the
  status lines), honors `gpg.minTrustLevel`, exits non-zero on
  unsigned/bad/low-trust; `-v` writes the signed payload to stdout.
- `log.rs`: `%G?`/`%GS`/`%GK`/`%GF`/`%GP`/`%GT`/`%GG` (good-but-untrusted ->
  `U`; unknown `%G<x>` and bare `%G` pass through). `show.rs`:
  `--show-signature`/`--no-show-signature` + `log.showSignature`.
