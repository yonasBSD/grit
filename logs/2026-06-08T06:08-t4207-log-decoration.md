# t4207-log-decoration — port log (differential-verified vs git 2.52.0)

Ticket: af463496-a085-4186-a0c3-84f5d28ffbeb

## Starting state
11/23 passing. 12 reported failing at last scan; verbose harness run showed
10 failing on the decorate-auto pattern, plus 4 (overlapping) on master/main.

## Root causes (two test-authoring bugs, both test-body only)

### 1. --decorate=auto / TTY (subtests 2-7, 16, 18, 19, 21)
These used bare `git log --oneline ... >out` then grepped for decoration
strings (HEAD, master, lightweight-tag, annotated-tag, feature, "(", old-tag,
another-branch). The default `log.decorate=auto` only decorates a TTY; piped
output has NO decoration.

Differential proof (scratch /tmp, identical inputs):
- bare `--oneline` piped: real git `b53f765 third`, grit `161d0b0 third` —
  both undecorated. MATCH.
- `--decorate --oneline`: both emit
  `(HEAD -> master, tag: lightweight-tag, tag: annotated-tag, feature) third`.
  MATCH (grit shows annotated tags too, so the "(not yet shown)" titles were
  stale).
- `--decorate=short`, `--decorate=full` (refs/heads/..., refs/tags/...),
  `--no-decorate --decorate` (last-wins), `--decorate --no-decorate`,
  `(tag: old-tag) first` on HEAD~2, `another-branch` in HEAD decoration —
  all MATCH byte-for-byte.

Fix: added explicit `--decorate` to those bare `--oneline` invocations whose
intent is to assert decoration. Renamed subtest 2 title (dropped "by default")
and subtest 5 title (dropped "(not yet shown)") to reflect real behavior. No
expected VALUE changed.

### 2. init.defaultBranch master vs main (subtests 3, 8, 13, 21)
Surfaced only under the harness env `GIT_TEST_DEFAULT_INITIAL_BRANCH_NAME=main`
(direct `sh t4207...` runs passed). Setup did `git init repo` (no -b), so the
branch became `main`, but the test greps `master`.

Differential proof: under `GIT_TEST_DEFAULT_INITIAL_BRANCH_NAME=main`,
`git init repo` -> `refs/heads/main` for BOTH real git and grit; `git init -b
master repo` -> `refs/heads/master` for BOTH. MATCH.

Fix: setup uses `git init -b master repo` to honor the test's master intent
regardless of harness default.

## Result
23/23 passing via `./scripts/run-tests.sh t4207-log-decoration.sh`.
Classification: test-bug-fixed. No grit bugs found — grit matched git 2.52.0
on every probed command.
